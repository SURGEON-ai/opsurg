Installer l'app sur iPhone
  1. Ouvrez cette page dans Safari sur votre iPhone
  2. Touchez le bouton Partager (carré avec flèche vers le haut)
  3. Faites défiler et touchez « Sur l'écran d'accueil »
  4. L'app OpSurg apparaît comme une icône native

Sur iPhone, utilisez le micro du clavier iOS (bouton micro à côté de la barre d'espace) pour une dictée médicale française de qualité supérieure. Le bouton micro intégré utilise la Web Speech API (excellente sur Chrome PC).
